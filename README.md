# AI Voice Caller – Lead Qualification Agent
- A browser-based AI voice agent that can call, listen, respond, and qualify leads in real-time.
- Built as a lightweight prototype inspired by modern voice AI systems.

## What it does
- Initiates a simulated voice call
- Speaks to the user using Text-to-Speech
- Listens using Speech-to-Text
- Asks intelligent questions
- Qualifies users as:
  <ul>HOT LEAD</ul>
  <ul>COLD LEAD</ul>   

### Use Case
- This mimics real-world scenarios like:
- Lead qualification calls
- Customer support automation
- Sales outreach automation

### How It Works
= AI starts the conversation
- User responds via microphone
- Speech → converted to text
- Text → sent to backend
- AI generates response
- Response → converted to voice
- Loop continues until lead is classified
